# Shinigami Game
Defense survival game

## Play

**https://www.shinirealms.xyz/ShinigamiGame**

## What's in the game

- Endless waves of enemies that keep getting harder
- Different characters
- Different realms
- Connect your Solana wallet (Phantom, Solflare, etc.)
- Earn Soul Orbs and climb the leaderboard
- Your progress saves to the cloud

## Controls

- WASD to move
- Left Click / Right Click to attack
- E to interact
- C for power up
- LShift to dash
- ESC to pause
- On mobile theres 2 joysticks
